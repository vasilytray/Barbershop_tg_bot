# Сервис для записи через Телеграм бот (ТГ) на услугу к Мастеру. 

Разрабатывается под проект конкретного Barbershop, но фактически применим для любого проекта требующего записи на услугу.

Весь проект состоит из нескольких Докер-контейнеров с микросервисами, использующими свою докер-сеть:

1. Контейнер с Базой данных PostgreSQL для базы приложения
2. Контейнер с Базой данных для Планировщика задач
3. Контейнер с FrontEnd - лицо приложения на строне клиента.
4. Контейнер с BackEnd приложения реализует бизнес-логику на стороне сервера (FastAPI)
5. Вспомогательные контейенеры для мониторинга работы приложения.
